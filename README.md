### How to Start:
Install npm modules
```
npm install
```

Run application:
```
npm start
```

Navigate to localhost:3000/ and the application should be running.
Enjoy!

### How to Test:

Install npm modules
```
npm install
```

Run application:
```
npm start
```

Navigate to localhost:3000/tests and it should run through the tests
that have been created in tests.js.
